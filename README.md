# cmonkey - Highly Customisable Typing TUI with MonkeyType wordbank support
## (WIP - Initial phase)


cmonkey is a blazingly fast and configurable typing TUI with custom theming and support for ALL languages (via monkeytype's wordbanks)
Written in C11 without any libraries (writes raw ansi escape codes)
